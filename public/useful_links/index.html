Here I’ve collated useful links that may come in handy.

Finding papers:

  • NASA Astronomical Data System - a search engine for finding papers
  • arxiv - open access repository for the latest astronomy papers, beware - some of these mightn’t have been refereed yet!
  • Dawes reviews - a series of papers reviewing topical areas of astronomy comissioned by the Publications of the Astronomical Society of Australia.

Galaxies:

  • Emission line list - what’s that line? it’s one of these.
  • Cosmology calculator - get information for objects based on redshift and chosen cosmology
  • Nasa Extragalactic Database - everything you need to know about an object.
  • Staralt - object visibility plotter

  • Matplotlib colours
